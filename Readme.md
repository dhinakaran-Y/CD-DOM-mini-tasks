   ### Task 3: Dynamic User Manager
    
Logic:
- On clicking "Add", dynamically create a div card showing the Name and Job.
- Constraint: If the Job Title is "Developer", add a specific CSS class (e.g., green background) to that card.
- Interaction: Every card must have a "Delete" button. Clicking it should remove only that specific card from the DOM.

 ### Task 4: The "Active State" Navigator
    
Logic:
- UI: A Navigation Bar with 5 links (Home, About, etc.).
- Loop through them using forEach to attach a click event.
- Interaction: When a link is clicked, add an .active class (change color/style) to the clicked link and remove it from all others.
